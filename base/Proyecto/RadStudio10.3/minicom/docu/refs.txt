// Falta completar: COMSPEC=, SHELL=, OSTYPE=, TMPPATH, TERM=linux

 -- Usos --
 Minicom consta de cuatro comandos internos: ayuda, lengua, cmd y salir.
 » cmd dir
 » ayuda	-->> muestra una ayuda explicativa en pantalla.
 » cmd cmd /?
 » cmd start explorer .
 » ^Z
 » ^L		-->> borra la pantalla.
 » ^D
 » salir	--> sale de minicom.
 » lengua	--> informa la config. actual de minicom en cuanto a idioma/región
		    este comando es similar a 'locale'.

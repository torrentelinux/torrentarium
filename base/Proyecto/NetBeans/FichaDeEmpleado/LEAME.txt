[+] Programación Java
Proyecto "FichaDeEmpleado" para Apache NetBeans IDE, Windows/Linux.
Ejercicio de programación Java + SQL + JDBC.
Puede ejecutar desde la línea de comandos la aplicación .jar:
    C:\base\Proyecto\NetBeans\FichaDeEmpleado>java -jar dist\FichaDeEmpleado.jar
o también:
    C:\base\Proyecto\NetBeans\FichaDeEmpleado>javaw -jar dist\FichaDeEmpleado.jar

En el directorio sql\ se encuentran los archivos SQL necesarios para subirlos al
servidor SQL que corresponda:
    persona-mysql.sql		para importar al servidor MySQL
    persona-postgresql.sql	para importar al servidor PostgreSQL

Contiene la definición de la tabla 'empleado' de la base de datos 'persona'.

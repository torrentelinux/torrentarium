- Numeracion Binaria - Edición Windows -

Su objetivo es servir de plataforma educativa para la enseñanza del sistema de numeración binaria
tanto en Windows como en Linux.
Fijarse en las imágenes gráficas que acompañan a esta documentación para conocer sus opciones de uso.
Para empezar a usar, abra la consola de textos de su S.O. y escriba:
    java -jar dist\NumeracionBinaria.jar	(Windows/cmd)
    java -jar dist/NumeracionBinaria.jar	(Linux/bash)

El Proyecto Java 'NumeracionBinaria' ha sido realizado en Apache NetBeans IDE.
Software educativo para sistemas Windows y Linux.

::HOSPEDAJE::
El proyecto 'NumeracionBinaria' está hospedado en GitHub, visite el siguiente enlace:
	https://github.com/torrentelinux/torrentarium/tree/master/base/Proyecto/NetBeans/NumeracionBinaria

Allí encontrará el paquete .zip que corresponde a la versión para Windows.
Haga clic sobre el paquete "NumeracionBinaria-windows-1.6.2026.zip" y luego clic sobre el botón 'Download raw file' (a la derecha)
y el navegador procederá a descargar el paquete seleccionado.

::INSTALACION::
Se ofrece dos paquetes de archivos según sea el S.O. que Usted use diariamente.
Paquete completo para Linux:
	NumeracionBinaria-linux-1.6.2026.tar.xz		{1}
Paquete completo para Windows:
	NumeracionBinaria-windows-1.6.2026.zip		{2}

Para el caso {2}, no es necesario que Usted sea 'Administrador' o que tenga privilegios altos para instalar 'NumeracionBinaria'.
Descomprima todo en un directorio, público o personal, y pase adentro de ./NumeracionBinaria/
En el mencionado directorio contiene lo siguiente:
NumeracionBinaria/.
├── build
│   ├── classes
│   │   └── numeracionbinaria
│   ├── empty
│   └── generated-sources
│       └── ap-source-output
├── dist			=> el archivo .jar principal
├── doc				=> la documentación del proyecto más la licencia GNU/GPL v3
├── nbproject			=> los archivos del proyecto para el entorno NetBeans
│   └── private
└── src
    └── numeracionbinaria	=> el archivo .java fuente

::IMPORTANTE::
Usted debe contar con conocimientos básicos de S.O. Windows para comenzar a desarrollar aplicaciones Java.
Le suguiero que busque info., docu., videos, etc. en Internet sobre Windows.

Se requiere la librería Java/JDK y/o Java/JRE para su correcto funcionamiento del proyecto 'NumeracionBinaria'.
El código fuente de "NumeracionBinaria.java" es compatible desde la versión 21 en adelante.
Si en su computadora no tiene instalada la librería mencionada, le sugiero que lea el siguiente artículo:
    - Cómo instalar Java (JDK) en Windows paso a paso -
    https://mundobytes.com/como-instalar-java-jdk-en-windows/

Si quiere utilizar Apache NetBeans IDE para aprender/estudiar/recompilar el código fuente de 'NumeracionBinaria'
tendrá que proceder a descargar e instalar el entorno de programación Java desde el sitio oficial:
    https://netbeans.apache.org/front/main/index.html

Y presionar el botón 'verde' que dice 'download' y lo llevará a otro enlace, es de 'codelerity'
    https://github.com/codelerity/netbeans-packages/releases/download/v30-build1/Apache-NetBeans-30.exe

Tocar sobre el botón que dice 'Apache NetBeans 30 + JDK 26 (.exe x86_64)' y procederá a descargar el paquete instalador.

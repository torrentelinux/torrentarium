- Numeracion Binaria - Edición Linux -

Su objetivo es servir de plataforma educativa para la enseñanza del sistema de numeración binaria
tanto en Linux como en Windows.
Fijarse en las imágenes gráficas que acompañan a esta documentación para conocer sus opciones de uso.
La aplicación requiere que corra en un ambiente gráfico porque ha sido vinculado en el proceso de compilación
con la librería javax.swing, el ambiente gráfico puede ser KDE, GNOME o cualquier otro que inicialize la interfaz gráfica.

Para empezar a usar, abra la consola de textos de su S.O. y escriba:
    java -jar dist/NumeracionBinaria.jar	(Linux/bash)
    java -jar dist\NumeracionBinaria.jar	(Windows/cmd)

O también, de esta manera: ./NumeracionBinaria
es un programa ejecutable en Bash que invoca a Java para que se inicie la aplicación 'NumeracionBinaria.jar'.

El Proyecto Java 'NumeracionBinaria' ha sido realizado en Apache NetBeans IDE.
Es software educativo para sistemas Linux y Windows. Se acompaña el archivo de la licencia GNU/GPL v3 'LICENSE'.

::INSTALACION::
Se ofrece dos paquetes de archivos según sea el S.O. que Usted use diariamente.
Paquete completo para Linux:
	NumeracionBinaria-linux-1.6.2026.tar.xz		{1}
Paquete completo para Windows:
	NumeracionBinaria-windows-1.6.2026.zip		{2}

Para el caso {1}, no es necesario que Usted sea 'root' o que tenga privilegios altos para instalar 'NumeracionBinaria'.
Descomprima todo en un directorio, público o personal, y pase adentro de ./NumeracionBinaria/
En el mencionado directorio contiene lo siguiente:
NumeracionBinaria/.
├── build
│   ├── classes
│   │   └── numeracionbinaria
│   ├── empty
│   └── generated-sources
│       └── ap-source-output
├── dist			=> el archivo .jar principal
├── doc				=> la documentación del proyecto, las imágenes gráficas más la licencia GNU/GPL v3
├── nbproject			=> los archivos del proyecto para el entorno NetBeans
│   └── private
└── src
    └── numeracionbinaria	=> el archivo .java fuente

::IMPORTANTE::
Se requiere la librería Java/JDK y/o Java/JRE para su correcto funcionamiento.
El código fuente de "NumeracionBinaria.java" es compatible desde la versión 17 en adelante.
Visite el sitio de OpenJDK, consulte allí lo que hace falta para instalar en su S.O. Linux.
También puede buscar info. en el mismo sitio web de su sistema Linux para disponer de Java.
Otra alternativa es fijarse en el sitio web de Apache NetBeans y descargar el IDE junto
con la librería Java/JDK para su sistema Linux.

::PROCESO DE DESARROLLO::
Puede trabajar desde la consola de textos del S.O. ya sea editando, compilando y ejecutar el archivo .jar
pero es más cómodo hacer todo eso desde el entorno de NetBeans.
Linux cuenta con numerosos editores de textos para programadores, por ejemplo, mcedit, kwrite, xed, gedit, etc.
Los comandos de Java para el proceso de desarrollo son: javac, jar y java.
Cada uno de ellos cuenta con ayuda a través del comando man, por ejemplo, man javac.

Compilar .java: javac src\numeracionbinaria\NumeracionBinaria.java
Ejecutar .class: java -cp src numeracionbinaria.NumeracionBinaria
Ejecutar .jar: java -jar dist\NumeracionBinaria.jar

Envío de mensajes de texto desde una computadora hacia un teléfono celular.
Comunicación a la API de WhatsApp mediante lenguaje Java.

Ejecuta el proceso 'arg' y espera su finalización
    int estado = new ProcessBuilder(arg).inheritIO().start().waitFor();

Navegadores de páginas HTML que pueden ser invocados desde ProcessBuilder():
En Windows:
    String aplic = "C:\\Program Files (x86)\\Slimjet\\slimjet.exe";
    String aplic = "C:\\Program Files\\Mozilla Firefox\\firefox.exe";

En Linux:
    String aplic = "/usr/bin/firefox";

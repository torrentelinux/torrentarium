    Técnicas de Programación aplicadas al desarrollo de la librería JLogo

Investigación y Desarrollo básico en Java de la Tortuga.
Se utilizó el sistema operativo Oracle Linux Server 7.9 y las herramientas de
desarrollo OpenJDK 16.0.1+9
La programación se efectuó en Apache NetBeans IDE 25, OpenJDK 23.0.2+7
Todo el software estuvo basado en 64 bits.

Borra toda la pantalla - Take Command - JPSoft:
· setdos /a1 -- Activa caracteres ANSI
· echo ^e[2J  -- Borra la pantalla en tccle14x64 - Take Command v14 - ANSI X3.64 Command Reference -
· crear directorios \base\programa en c:
· definir var. de entorno 'base' ==> set base=c:\base
· crear archivo 'bp' en '%base%\programa' --> echo ^e[2J > bp
· type %base%\programa\bp  ==> borra toda la pantalla

Borra toda la pantalla - Command cmd.exe - Microsoft Windows:
· Para introducir el código ESC (0x1B) presione las teclas 'Ctrl' y '[' y da como resultado '^['
· echo ^[[2J  -- Borra la pantalla en 'cmd - Microsoft Windows'
· crear directorios \base\programa en c:
· definir var. de entorno 'base' ==> set base=c:\base
· crear archivo 'bp' en '%base%\programa' --> echo ^[[2J^[[1;1H > bp
· type %base%\programa\bp  ==> borra toda la pantalla y pone el cursor en la fila 1, columna 1.
